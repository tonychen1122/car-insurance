機車族｜保障缺口

強制險只是底線
這 4 個保障你才真的有保到

很多機車族有保險，但出事才發現根本不夠用

三個強制險保不到的缺口
缺口 1

自己受傷
強制險沒你的份

強制險是保「被你撞的人」，騎車的你本人受傷、骨折、住院，強制險一毛不賠。

→ 補：駕駛人傷害險
缺口 2

撞傷對方超過 20 萬
超出的你要自賠

強制險傷害醫療上限 20 萬。對方若重傷或住院過久，超出部分你要自己想辦法。

→ 補：第三人責任險
缺口 3

住院期間
收入直接歸零

受傷不能工作，健保只解決部分醫療費。住院差額、看護費、薪水空窗期全沒人補。

→ 補：醫療險 + 意外險
四張必補保單，各自補什麼
🛡️
駕駛人傷害險
保「騎車的你」本人
強制險不保騎車者本身，這張保單專門填補這個空缺。
  • ✓ 騎車意外造成自身身故
  • ✓ 意外失能（殘廢）給付
  • ✓ 意外傷害醫療費用
  • ✗ 疾病住院（非意外不保）
⚖️
第三人責任險
補強「賠別人」的上限
強制險最高賠 20 萬，超出部分由這張保單接手，避免一場事故掏空存款。
  • ✓ 對方身體傷害超出強制險上限部分
  • ✓ 對方財物損失賠償
  • ✓ 法律訴訟相關費用
  • ✗ 自身傷亡（需搭配駕駛人傷害險）
🏥
醫療險
補健保不足的住院費用
健保不能完全解決住院費用，醫療險補足住院差額、手術費等自費項目。
  • ✓ 住院病房差額補貼
  • ✓ 手術費用理賠
  • ✓ 意外或疾病住院均可理賠
  • ✗ 收入損失補貼（需另保意外險）
💰
意外險
補住院期間薪水空窗
受傷不能上班，每天仍有住院日額補貼，讓你不用邊養傷邊擔心收入中斷。
  • ✓ 意外住院日額給付（每日補貼）
  • ✓ 意外身故 / 失能保險金
  • ✓ 骨折不住院也有給付
  • ✗ 疾病住院（僅限意外）

⚠ 注意事項
① 強制險傷害醫療給付上限為 20 萬元（2024 年現行規定），超出部分須自行負擔。
② 駕駛人傷害險屬產險附加險種，需與強制險或第三人責任險搭配投保。
③ 醫療險與意外險屬壽險範疇，保費與保額依個人年齡、健康狀況、保險公司而異。
④ 各保險公司條款細節不同，實際理賠以投保保單條款為準，建議詳閱條款或諮詢專業人員。
